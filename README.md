# 📈 Stock Market Intelligence Dashboard

An interactive AI-powered Stock Market Intelligence Dashboard built using Python, Streamlit, Plotly, and Yahoo Finance.

The application allows users to analyze stock performance, visualize technical indicators, monitor portfolio metrics, and explore market trends through an intuitive web interface.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mondals472003-bit/stock-market-intelligence-dashboard.git
```

Move into the project directory:

```bash
cd stock-market-intelligence-dashboard
```

Create virtual environment:

```bash
python -m venv .venv
```

Activate virtual environment:

### Windows

```bash
.venv\Scripts\activate
```

### Linux / Mac

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Locally

Start the Streamlit application:

```bash
streamlit run app.py
```

Open:

```text
http://localhost:8501
```
