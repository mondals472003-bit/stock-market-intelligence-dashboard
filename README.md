# Stock Market Intelligence Dashboard

## Overview

An end-to-end stock market intelligence platform built using Python that combines technical analysis, sentiment analysis, portfolio analytics, and machine learning-based stock forecasting.

## Features

- Historical stock data analysis
- SMA20 and SMA50 indicators
- RSI analysis
- MACD analysis
- Buy/Sell signal generation
- Financial news sentiment analysis using FinBERT
- Portfolio performance analytics
- XGBoost stock price forecasting

## Technologies

- Python
- Pandas
- Plotly
- Streamlit
- FinBERT
- XGBoost
- Scikit-Learn

## Model Performance

- R² Score: 0.844
- MAE: 21.24
- RMSE: 27.96
